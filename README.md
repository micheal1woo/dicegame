## 🎲 주사위 게임


### 던지기 버튼
 - 나와 상대의 주사위를 동시에 굴린다 
### 처음부터 버튼
 - 나와 상대의 주사위를 동시에 리셋한다

### 기능
 - 주사위 굴리기
 - 주사위의 총합 계산
 - 주사위게임의 히스토리 표시
